## EDA - Students Social Media Addiction (Python Project)

Project Overview :-

This project performs Exploratory Data Analysis (EDA) on student social media usage data to identify addiction patterns and their impact on:

🧠 Mental Health

😴 Sleep Patterns

📚 Academic Performance

📱 Social Media Behavior

The goal is to transform raw real-world data into meaningful insights using Python.

## Key Insights :- 

- High social media usage increases addiction
- Poor sleep is linked to high usage
- Students using >5 hours/day show academic decline
- WhatsApp, TikTok, Snapchat show higher addiction
